个人博客搭建

利用 Github 开源库搭建免费的个人博客

JYDay’ Blog

啊真的是，将近一天的功夫才弄好这个东西（终归还是我太菜了趴…），也不知道看了有多少的博客教学啊什么详解啊，又学了半天的md语法啊……反正我是那个迷茫啊！！然后看这个看那个的慢慢摸索出来的……

以下是我整理的个人感觉是挺详细的一套流程了吧…..（没错！就素介个亚子！）

准备工作及本地模板

嘛！按照步骤来啦~

1、在Github官网创建一个账号 –> Github官网

之后有个验证真人还是机器人…Emmm

有免费和收费两种，不是很经常用到 Github 的免费就可以了（反正我选的是免费的…）,大佬随意哈！！

ps：验证QQ邮箱接收验证码不一定能接收到（应该不是我吧…），建议用手机自带的浏览器打开这个网站进行注册和接收（绝无虚言，本人亲测！！）

ok！nice！以后你就是有 github 账号的人啦 2333

之后就可以创建你的 Github 的仓库啦~（就是利用 Github 的开源库搭建的博客哦）

点击右上角的加号，选择第一个 New repository

填写库名信息，注意前缀一定要跟你的用户名一样哦

ok，创建成功，然后就有了一个用来搭建博客的仓库了

2、安装 Git –> Git下载官网
自己修改位置安装咯，安装完成后点击鼠标右键会有 Git Bash Here 的命令行工具选项，以后就用这个来使用 git 啦
命令 git –version 查看版本信息

Git 是目前世界上最先进的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理，也就是用来管理你的 hexo 博客文章，上传到 GitHub 的工具，Git 非常强大，建议每个人都去了解一下哦

3、安装 node.js –> node.js下载官网
Hexo 是基于 nodeJS 编写的，主要还需要用到里面的 npm 工具

因为本文讲的是 windows 下的安装，选择 LTS 下的 windows 版本的就ok

4、安装 Hexo
Hexo 是一个快速、简洁且高效的博客框架，Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页

直接使用 npm 命令安装 Hexo
(建议安装在博客目录下，新建文件夹 eg：BoKe/hexo 或者 Blog/hexo 等)
在 hexo 文件下右键然后选中 Git bash Here 打开命令栏

cd 的作用是传送到你安装 hexo 的目录，如果之前就是在此目录打开则不需要再次输入

hexo init 安装到此目录，安装完成如下

欧克！本地的模板就大功告成啦！

之后你的 public 文件夹就会生成相关的 html 等文件啦

hexo s 是开启本地预览服务，打开浏览器访问 http://localhost:4000 即可看到你的博客的内容啦，类似下图(一直偷图会不会不太好……)
(默认端口为：4000 ，多人会碰到浏览器一直在转圈但是就是加载不出来的问题，一般情况下是因为端口占用的缘故)

ok，至此我们的本地博客模板搭建成功！

本地模板与 Github 库连接

与 Github 连接当然需要一个 Github 账号和 Github 库啦

这些在准备工作都已经做好了哦

回到你的 git bash中~

这里的 yourname 输入你的 GitHub 用户名，youremail 输入你 GitHub 的邮箱，这样 GitHub 才能知道你是不是对应它的账户

输入完之后可以用以下两条检查一下你有没有输对

然后创建 SSH，生成与 Github 连接的秘钥，这个按默认一路回车就可以了

这个时候它会告诉你已经生成了 .ssh 的文件夹，在你的电脑中找到这个文件夹

ssh ，简单来讲，就是一个秘钥，其中，id_rsa 是你这台电脑的私人秘钥，不能给别人看的，id_rsa.pub 是公共秘钥，可以随便给别人看，把这个公钥放在 GitHub 上，这样当你链接 GitHub 自己的账户时，它就会根据公钥匹配你的私钥，当能够相互匹配时，才能够顺利的通过 git 上传你的文件到 GitHub 上

之后就要打开 Github 官网 登录配置啦

点击右上角的个人，然后选项 Settings

点击左侧栏 SSH and GPG keys 选项

在 New SSH keys 的 key 中输入秘钥（id_rsa.pub 的全部内容），title 随便输入

然后点击 Add SSH key ，就添加成功啦

ok，添加后在 Git Bash 中查看是否成功

然后呢，打开你的 hexo 的文件夹，找到其中的配置文件（ _config.yml ）打开并翻到最后，修改 deploy 的内容

YourgithubName 就是你的 GitHub 账户，注意每个冒号后面有一个空格

这个时候需要先安装 deploy-git ，也就是部署的命令,这样你才能用命令部署到 GitHub

然后呢

（hexo d 之前可以先 hexo s ，打开本地预览在 http://localhost:4000 可以本地先查看确认无误在部署哦）

成功之后会需要一点时间才能打开，大概五分钟（不等）之后就可以在http://yourname.github.io 这个网站看到你的博客啦！

专属域名与博客个性主题

1、专属域名设置

现在我们的个人博客网站地址是 http://yourname.github.io ，如果觉得这个网址‘逼格’不够，就要我们自己设置域名啦

可以在 腾讯云、阿里云 等地方购买一个域名，看个人喜好选后缀，价格不一，也不是很贵

（本人这个就在腾讯云买了后缀为 .club 的域名，新人，一年一块钱…）

以腾讯云为例，购买域名，然后按照要求完成实名认证

对域名进行解析

增加两条域名解析：

如果不知道自己网站的 ipv4 地址呢，就打开 cmd 命令行(同时按下 windows 键和 R 键，输入 cmd )，输入 ping 自己的网址，就可以得到回复啦

ok，添加之后保存，过一会就生效咯

打开 hexo 文件的 source 文件夹，在里面创建一个名为 CNAME 的文件
注意：文件名一定不能错，且不能有后缀
在里面输入你购买的域名（比如我的就是 jyday.club 啦）

之后回到 Github 官网 ，到项目的 Settings 选项
往下滑，找到 Custom domain ，在里面添加你购买的域名并保存

ok，这样就可以以你购买的域名来访问你的博客啦！比如我的就是 https://jyday.club
