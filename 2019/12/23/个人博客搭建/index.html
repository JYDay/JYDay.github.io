利用Github搭建个人的免费博客

JYDay’ Blog

啊真的是，将近一天的功夫才弄好这个东西(终归还是我太菜了趴…)，也不知道看了有多少的博客教学啊什么详解啊，又学了半天的md语法啊……反正我是那个迷茫啊！！然后看这个看那个的慢慢摸索着个人感jio是最便捷的方法了吧……

准备工作

嘛！按照步骤来啦~

1、在Github官网创建一个账号 –> Github官网


之后有个验证真人还是机器人…Emmm
有免费和收费两种，不是很经常用到Github的免费就可以了(反正我选的是免费的…),大佬随意哈！！
ps：验证QQ邮箱接收验证码不一定能接收到(应该不是我吧…)，建议用手机自带的浏览器打开这个网站进行注册和接收(绝无虚言，本人亲测！！)
ok！nice！以后你就是有github账号的人啦2333？

之后就可以创建你的Github的仓库啦~(就是利用Github的开源库搭建的博客哦)
点击右上角的加号，选择第一个 New repository

填写库名信息，注意前缀一定要跟你的用户名一样哦

ok，创建成功，然后就有了一个用来搭建博客的仓库了

2、安装Git –> Git下载官网
自己修改位置安装咯，安装完成后点击鼠标右键会有 Git Bash Here的命令行工具选项，以后就用这个来使用git啦
命令 git –version 查看版本信息

Git是目前世界上最先进的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理，也就是用来管理你的hexo博客文章，上传到GitHub的工具，Git非常强大，建议每个人都去了解一下哦

3、安装node.js –> node.js下载官网
Hexo 是基于nodeJS编写的，主要还需要用到里面的npm工具
因为本文讲的是windows下的安装，选择LTS版本就ok
